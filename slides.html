Scanner de PDF

Projeto de Leitura e Busca por Palavra-chave

Desenvolvido em linguagem C

Equipe: Ramom Souza, Eduardo Molica, Antonio Ferreira, Yuri Bom

Objetivo do Projeto

  • Receber o caminho de uma pasta com arquivos .txt e .pdf
  • Perguntar ao usuário uma palavra-chave
  • Contar quantas vezes a palavra aparece em cada arquivo .txt
  • Exibir os PDFs relacionados, ordenados por quantidade de ocorrências

Bibliotecas Utilizadas

  • <stdio.h>
  • <stdlib.h>
  • <string.h>
  • <unistd.h>

Estrutura de Variáveis

  • path: manipula path.txt
  • file, file2, file3: leitura de arquivos txt e pdf
  • caminho[100]: caminho da pasta
  • comando[100]: comandos a serem executados no CMD
  • palavra[100]: palavra-chave da busca

Listagem de Arquivos

  • Gera dois comandos:
  • Listar todos os arquivos .txt → dir /s /b "*.txt" > listaTXT.txt
  • Listar todos os arquivos .pdf → dir /b "*.pdf" > listaPDF.txt
  • Executa os comandos com system()

Remoção de Arquivo Temporário

O arquivo path.txt é apenas temporário.

Depois de usado, o programa o exclui com:

system("del /f path.txt");

Contagem de Arquivos

Conta quantas linhas tem no listaTXT.txt.

Isso indica quantos arquivos .txt precisam ser processados.

Recebendo a Palavra-chave

  • Usuário digita a palavra
  • Loop de validação para evitar busca com string vazia
  • Remove o \n final de fgets

Busca por Palavra-chave

  • Abre cada arquivo .txt
  • Lê linha por linha, caractere por caractere
  • Conta as ocorrências da palavra-chave
  • Armazena o resultado no vetor ocorrencias[]

Vetor de Resultados

Criação de dois vetores paralelos:

  • nomesArquivos[100][100]: nomes dos arquivos PDF
  • ocorrencias[100]: quantidade de ocorrências da palavra

Ordenação dos Resultados

Implementação de um Selection Sort:

  • Compara todos os elementos com os seguintes
  • Troca de posições para colocar em ordem crescente

Animação de Espera

Durante o processamento:

sleep(1);

Simula progresso para o usuário com pontos: .

Exibição Final

  • Mostra nome de cada PDF e o número de ocorrências
  • Informa o total de arquivos onde a palavra foi encontrada

Conclusão

O programa permite uma busca rápida por palavras-chave dentro de lotes de arquivos .txt, mostrando os PDFs relacionados.

Exemplo prático: Filtrar documentos por tema específico.
